## Sobre o Projeto

Este projeto foi desenvolvido como parte do bootcamp **CI&T - Do Prompt ao Agente** oferecido pela plataforma **DIO (Digital Innovation One)**. 

O grande diferencial desta aplicação é que **toda a sua estruturação, lógica e codificação foram geradas utilizando o Agente GitHub Copilot no VS Code**. O projeto serviu como laboratório prático para aplicar técnicas avançadas de engenharia de prompt e desenvolvimento assistido por agentes de IA.

Trata-se de uma aplicação web interativa e responsiva que calcula a pegada de carbono de diferentes trajetos pelo Brasil. O objetivo é conscientizar os usuários sobre o impacto ambiental de suas escolhas de transporte, oferecendo comparativos visuais e estimando o custo para compensação através de créditos de carbono.
## Funcionalidades

* **Autocompletar de Rotas:** Base de dados local com rotas populares brasileiras que preenche a distância automaticamente.
* **Cálculo por Modal:** Compara emissões entre Bicicleta, Carro, Ônibus e Caminhão.
* **Comparativo Visual:** Barras de progresso dinâmicas e codificadas por cor para ilustrar a eficiência de cada meio de transporte em relação ao carro.
* **Créditos de Carbono:** Estima quantos créditos são necessários para compensar a viagem e o valor financeiro médio.
* **UX/UI Aprimorada:** Feedback visual de carregamento (spinner) e scroll suave para os resultados.

## Tecnologias e Práticas Utilizadas

* **HTML5:** Estrutura semântica e formulários acessíveis.
* **CSS3:** Estilização responsiva (Mobile First), utilizando **Metodologia BEM** e *CSS Custom Properties*.
* **JavaScript (Vanilla):** Arquitetura modular separada em objetos globais (`RoutesDB`, `CONFIG`, `Calculator`, `UI`).
* **CI/CD (GitHub Actions):** Deploy automatizado configurado via fluxo de trabalho YAML para publicação direta no GitHub Pages.
